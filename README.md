# 🔗 Malicious URL Detector

---

A Machine Learning project that detects whether a URL is **Safe** or **Malicious**.

This project uses Python and Machine Learning algorithms to analyze URLs and predict phishing or malicious websites.

Because clicking random links on the internet has somehow become a global hobby.

---

# 🚀 Features

- Detects malicious URLs
- Beginner-friendly project
- Fast prediction system
- Simple web interface
- Machine Learning based detection

---

# 🛠️ Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- Streamlit

# ⚙️ How It Works

1. User enters a URL
2. URL features are extracted
3. Machine Learning model analyzes the URL
4. The model predicts whether the URL is:
   - Safe
   - Malicious

# 📦 Installation

## Step 1: Clone the Repository

```bash
git clone https://github.com/your-username/Malicious-URL-Detector.git
```

## Step 2: Go to the Project Folder
```bash
cd Malicious-URL-Detector
```

## Step 3: Install Required Libraries
```bash
pip install -r requirements.txt
```

## Step 4: Run the Application
```bash
streamlit run app.py
```
